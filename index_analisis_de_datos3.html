Sobre el proyecto

¡Bienvenido al proyecto de análisis de datos de experimentos A/B! En este caso, analizaremos los datos de un sitio web que lanzó un experimento A/B con el objetivo de aumentar sus ingresos. Para analizar los resultados del experimento, utilizaremos herramientas como excel y R.

Se nos proporciona un archivo de Excel con datos brutos que incluyen el ID del usuario, el tipo de muestra y el ingreso generado por el usuario. El archivo contiene datos simulados de un experimento A/B con algunas características engañosas que pueden afectar el resultado final del experimento. Se nos solicita analizar si hay una diferencia significativa entre las dos opciones.

Acciones tomadas

  1. Definición del business task: Lo primero que se hizo fue examinar el input de los Stakeholders para determinar cuál era el business task.
  2. Estudio de los datos: La siguiente acción fue estudiar las propiedades de los datos (tipo y características) y su contexto (metadatos y origen). De esta forma se determinó que los datos era de calidad y que no contaba con bias.
  3. Preguntas SMART: Luego se definieron las preguntas SMART que se esperaban contestar a través de este proyecto de análisis de datos. Estas preguntas brindaron una guía de lo que se deseaba analizar.
  4. Preparación de los datos: Luego se procedió a preparar los datos asegurando que los datos estuvieran en el formato adecuado para trabajar con ellos posteriormente. Además, se filtraron y clasificaron los datos para una mejor limpieza en la siguiente fase.
  5. Limpieza y procesamiento de los datos: La siguiente actividad fue la limpieza de los datos donde se buscaron duplicados, datos nulos y errores. Luego se separaron y mezclaron datos donde se consideró adecuado.
  6. Análisis de datos: Con los datos lista para trabajar, se utilizaron distintas herramientas de análisis de datos, estadística y visualización de datos. Entre algunas se pueden mencionar: R, tableau y SQL. Luego se hicieron observaciones de los descubrimientos para posteriormente anexarlos en el reporte del proyecto.

Business task

La finalidad de este proyecto es la siguiente: analizar los resultados de un experimento A/B llevado a cabo en un sitio web con el objetivo de aumentar sus ingresos y proporcionar recomendaciones basadas en los resultados del análisis.

  • alfa >5%: Se observa que el 8.2% de las veces el resultado observado es motivo del azar, por lo cuál se concluyó que este resultado no fue estadísticamente significativo y se conserva la hipótesis nula.

Conclusión

El resultado obtenido sugiere que la diferencia observada en el ingreso generado entre el grupo control y el grupo variante esta dentro del rango de variación por azar, por lo tanto, es un resultado estadísticamente no significativo.
