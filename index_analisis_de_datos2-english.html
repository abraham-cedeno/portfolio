Section 1: Introduction to the Project



Problem Description

AdventureWorks, a global manufacturing company, has recognized the importance of data analysis in decision-making. The company requires a comprehensive business intelligence solution to monitor its key KPIs, such as sales, revenue, profits, and returns.

I have been provided with a set of raw CSV files to design and develop a solution using Power BI.

Business Objective

"Build an interactive dashboard with clear and easy-to-understand visualizations that allow users to track key KPIs, compare regional performance, analyze product trends, project outcomes, and identify high-value customers."

Section 2: Data Description

The data provided by AdventureWorks includes information on transactions, returns, products, customers, and territories in the form of raw CSV files. The data was collected directly from AdventureWorks and is updated periodically every month. It is used to monitor the company's key KPIs and adjust the business strategy accordingly.

Section 4: Building the Dashboard

After constructing the relational data model, establishing relationships between tables, and creating calculated columns and DAX measures, an interactive report was designed in Power BI. The report offers clear and understandable visualizations that allow users to deeply analyze the information and gain insights into the company's key KPIs.

The resulting interactive dashboard is a valuable tool for decision-making and allows users to quickly access relevant and up-to-date information.

Dashboard Contents


The dashboard has 3 sections:
  • Executive Summary
  • Product Details
  • Customer Details



    • Section 5: Conclusion

    This project demonstrates how Power BI can be used to develop a comprehensive business intelligence solution from a set of raw CSV files.

    Creating an interactive dashboard with clear and understandable visualizations allows users to make informed and accurate decisions based on the data available in the relational data model. The developed solution will help AdventureWorks monitor its key KPIs and continuously improve its business strategy.
